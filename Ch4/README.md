## 專有名詞

| 英文術語 | 中文翻譯 | 說明 |
|---------|---------|------|
| Assembler | 組譯器 | 將組譯語言轉換為機器碼的工具 |
| Assembly Language | 組譯語言 | 低階程式語言，使用助記符號表示指令 |
| Machine Code | 機器碼 | CPU 可直接執行的二進位指令 |
| Opcode | 運算碼 | 指令的操作碼部分 |
| Operand | 運算元 | 指令的操作對象 |
| Label | 標籤 | 用於標記位置的符號 |
| Symbol Table | 符號表 | 存放標籤與位址對應關係的資料結構 |
| Relocation | 重定位 | 調整程式碼位址的過程 |
| Linker | 連結器 | 合併多個目標檔案的工具 |
| Macro | 巨集 | 可展開為多條指令的代碼片段 |
| Directive | 指示 | 組譯器的控制指令 |
| Pass | 遍 | 組譯器的掃描次數 |
